# 单变量优化问题解答

## 问题：求解 min f(λ) = 2λ^2 - 5λ + 3，精度 δ = 0.1

### 1. 进退法确定包含最小点的区间

**目标**：找到区间 [a, b] 包含最小点。  
**步骤**：  
1. 初始点 λ0 = 0，步长 h = 0.5，f(0) = 3。  
2. 向前：  
   - λ1 = 0.5，f(0.5) = 2*0.5^2 - 5*0.5 + 3 = 1 < f(0)，继续。  
   - λ2 = 1.0，f(1.0) = 0 < f(0.5)，继续。  
   - λ3 = 1.5，f(1.5) = 2*1.5^2 - 5*1.5 + 3 = 0 > f(1.0)，停止。  
3. 区间：[1.0, 1.5]，包含最小点（f'(λ) = 4λ - 5，λ* = 1.25 ∈ [1.0, 1.5]）。

**答案**：区间 [1.0, 1.5]。

### 2. 0.618 法（黄金分割法）

**初始区间**：[a, b] = [1.0, 1.5]，精度 δ = 0.1。  
**步骤**：  
1. 初始：  
   - λ1 = 1.0 + 0.382*(1.5 - 1.0) = 1.191，f(1.191) ≈ -0.117。  
   - λ2 = 1.0 + 0.618*(1.5 - 1.0) = 1.309，f(1.309) ≈ -0.117。  
2. 迭代：  
   - 新区间 [1.191, 1.5]，长度 0.309 > 0.1。  
   - λ1 = 1.309，f(1.309) ≈ -0.117；λ2 = 1.382，f(1.382) ≈ -0.090。  
   - 新区间 [1.191, 1.382]，长度 0.191 > 0.1。  
   - λ1 = 1.264，f(1.264) ≈ -0.126；λ2 = 1.309，f(1.309) ≈ -0.117。  
   - 新区间 [1.191, 1.309]，长度 0.118 > 0.1。  
   - λ1 = 1.236，f(1.236) ≈ -0.124；λ2 = 1.264，f(1.264) ≈ -0.126。  
   - 新区间 [1.236, 1.309]，长度 0.073 < 0.1，停止。  
3. 结果：λ* ≈ (1.236 + 1.309)/2 = 1.2725，f(1.2725) ≈ -0.125。

**答案**：λ* ≈ 1.273，f(λ*) ≈ -0.125。

### 3. 牛顿法

**初始点**：λ0 = 1.5。  
**导数**：f'(λ) = 4λ - 5，f''(λ) = 4。  
**迭代公式**：λ_{k+1} = λ_k - (4λ_k - 5)/4 = 5/4 = 1.25。  
**步骤**：  
1. λ0 = 1.5，λ1 = 1.25，|λ1 - λ0| = 0.25 > 0.1。  
2. λ1 = 1.25，λ2 = 1.25，|λ2 - λ1| = 0 < 0.1，停止。  
3. 结果：λ* = 1.25，f(1.25) = -0.03125。

**答案**：λ* = 1.25，f(λ*) = -0.03125。

---

**总结**：  
- 进退法区间：[1.0, 1.5]  
- 0.618 法：λ* ≈ 1.273，f(λ*) ≈ -0.125  
- 牛顿法：λ* = 1.25，f(λ*) = -0.03125  
- 精确解：λ* = 1.25，牛顿法精确，0.618 法满足精度 δ = 0.1。
